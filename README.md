

🚀 Cyber Toolkit
A powerful, modular Cybersecurity Toolkit designed for learning, testing, and automating common security tasks including reconnaissance, analysis, monitoring, and exploitation techniques.

📌 Overview
Cyber Toolkit is a comprehensive platform built using Python and modern web technologies. It provides multiple cybersecurity utilities in a single interface, making it ideal for:
https://darkweb-monitor.up.railway.app

🚀 Installation
1️⃣ Clone the Repository
git clone https://github.com/your-username/Cyber-toolkit.git
cd Cyber-toolkit
2️⃣ Create Virtual Environment
python -m venv venv
source venv/bin/activate  # Linux/Mac
venv\Scripts\activate     # Windows
3️⃣ Install Dependencies
pip install -r requirements.txt
▶️ Running the Application
python app.py
Then open:

http://localhost:5000
